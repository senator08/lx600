git - система контроля версии
git hub - сервис\платформа\сайт, предоставляющий хранилище для версий кода
git clone ссылка на удаленный репозиторий - скопировать удаленный репозиторий 
git diff - просмотр изменений
git branch - вывод списка веток в терминал
git branch название_ветки - создание указанной название_ветки
git chekout название ветки - переход на указанную ветку
git switch название ветки 
git pull название связи название_ветки - стягивание изменений с удаленного репозитория, с указанной ветки
